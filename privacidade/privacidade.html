Política de Privacidade

1. Visão Geral

Esta Política de Privacidade descreve como coletamos, utilizamos e protegemos suas informações.
Ao utilizar nosso site, você concorda com estas práticas.
Nosso site não coleta informações diretamente de seus visitantes.
Todas as transações são realizadas por meio de comunicação direta com nossos colaboradores via WhatsApp, onde as informações necessárias para o processamento das operações são fornecidas pelos próprios usuários.

2. Informações Coletadas

As únicas informações coletadas são aquelas necessárias para realizar transações financeiras, como dados bancários fornecidos pelos usuários diretamente aos nossos colaboradores.
Esses dados são tratados com extrema confidencialidade, sendo eliminados após o término de cada transação.

3. Uso das Informações

As informações coletadas durante as transações são utilizadas exclusivamente para processar pagamentos e finalizar a operação de câmbio.
Não compartilhamos esses dados com terceiros.

4. Segurança dos Dados

Após a transação, todas as informações são excluídas, protegendo os dados do usuário de forma rigorosa contra acessos não autorizados ou compartilhamentos indevidos.
